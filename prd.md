# 产品需求文档 (PRD) - NotebookLM 水印移除 Web App

## 1. 项目概述
本项目旨在开发一个基于 Python 的 Web 应用程序，提供图形化界面，方便用户批量移除图片右下角固定的 "NotebookLM" 水印。

## 2. 目标
*   **可视化操作**：通过 Web 界面上传和管理图片。
*   **即时预览**：在保存前可对比原图与处理后的效果。
*   **批量处理与保存**：确认效果后，可将去水印后的图片批量保存到本地指定目录。

## 3. 功能需求

### 3.1 图片上传与选择
*   **多文件上传**：用户可以通过 Web 界面一次性选择并上传多张图片（支持 .png, .jpg, .jpeg）。
*   **文件列表**：显示已上传的图片列表。

### 3.2 水印移除与预览
*   **自动处理**：程序自动识别并尝试覆盖右下角水印。
    *   *核心逻辑*：读取图片右下角边缘的背景色，绘制矩形覆盖水印区域。
*   **交互式预览**：
    *   用户点击某张图片，界面显示“原图”与“去水印后”的对比图。
    *   提供简单的参数调整（可选）：如覆盖区域的大小、微调颜色容差（如果背景不是纯色）。

### 3.3 导出与保存
*   **下载/保存**：
    *   **方式 A (Web 标准)**：提供一个“下载全部”按钮，将处理后的图片打包成 ZIP 下载。
    *   **方式 B (本地应用)**：如果运行在本地环境，允许用户输入一个本地文件夹路径，点击“保存”直接写入磁盘。
    *   *当前侧重*：鉴于用户需求是“保存到指定位置”，我们将优先支持方式 B，但也提供 ZIP 下载作为备选。

## 4. 技术栈
*   **编程语言**：Python 3.x
*   **Web 框架**：**Streamlit** (推荐)
    *   理由：开发速度快，自带文件上传组件和图像展示组件，非常适合构建图像处理工具。
*   **图像处理**：Pillow (PIL)

## 5. 界面设计草图
1.  **侧边栏/顶部**：
    *   文件上传区 ("Drag and drop files here")
    *   目标保存路径输入框
2.  **主区域**：
    *   图片预览区：左右分栏显示 [原图] | [效果预览]
    *   控制区：如果有微调参数，放在这里。
    *   “执行并保存”大按钮。
